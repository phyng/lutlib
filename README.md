
# lutlib

A Python library for applying 3D LUTs to images.

## Installation

```bash
pip install lutlib
```

## Usage

```python
from lutlib import apply_lut

apply_lut("path/to/lut.cube", "path/to/image.png", "path/to/output.png")
```
